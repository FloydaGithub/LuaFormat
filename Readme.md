## Install

- Package Control

Search `LuaFormat` via `Package Control: Install Package`

- Git

git clone this repository to `Sublime Packages Path`

- Zip

download zip and unzip to `Sublime Packages Path`


## Usage

- shortcuts
`ctrl + alt + f`
